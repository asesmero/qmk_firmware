# VIA Asesmero 2x3 Layout

A simple layout with 6 keys with shorcuts to assign on OBS or any software
You can connect a Neopixel (RGB Led)on PIN F7 to see wich layer is active. 

LAYER 0 (RED)
+------------+------------+------------+
|    Ctrl    |    Ctrl    |    Ctrl    |
| + Shift    | + Shift    | + Shift    |
| + Alt + A  | + Alt + B  | + Alt + C  |
+------------+------------+------------+
|    Ctrl    |    Ctrl    |     TO     |
| + Shift    | + Shift    |    LAYER   |
| + Alt + D  | + Alt + E  |  1 (GREEN) |
+------------+------------+------------+
